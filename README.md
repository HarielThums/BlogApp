# BlogApp :smile::smile::smile:
 
## Projeto para estudo utilizando as tecnologias: 
-  Node.js
-  Express
-  MongoDB
-  Passport
-  Handlebars

Necessário instalar:
- Node.js
- MongoDB
- NPM

Iniciando aplicação:
- `npm i`
- `npm start`
